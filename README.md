# Bizagi

Prueba realizada para desarrollador iOS en Bizagi

## Estructura

Siguiendo las indicaciones dadas se realizó la prueba utilizando las mejores prácticas, de la misma forma se estructuró el proyecto.

nota: La estructura del proyecto puede cambiar al ser un proyecto grande, en este no esta una capa definida de persistencia y de io.
